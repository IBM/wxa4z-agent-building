# Deploy the zRAG Agent

- navigate back to `/stack`

- run `./deploy.sh up --profile upgrade-agent`


### Verify agent deployment 

- Verify agent logs using:

  `docker logs wxa4z-upgrade-agent-mcp-client`
  
  `docker logs wxa4z-upgrade-agent-mcp-server`



### Configure agent connection

- Navigate to `IBM wxa4z Management Console`
